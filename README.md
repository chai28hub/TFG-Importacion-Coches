#TFG: Sistema de Importación de Vehículos (Dubái & Alemania)

Este proyecto es una aplicación web dinámica diseñada para la gestión de un negocio de importación de coches de lujo desde Dubái y Alemania.

## 🛠️ Tecnologías utilizadas
* **Frontend:** HTML5, CSS3 (Diseño responsivo).
* **Backend:** PHP (Lógica de servidor).
* **Base de Datos:** MySQL (XAMPP).
* **Control de versiones:** Git & GitHub.

##Estructura de Datos
El sistema utiliza una base de datos relacional con las siguientes tablas:
1. **Coches:** Almacena el inventario disponible con filtros por origen.
2. **Clientes:** Registro automático de personas interesadas mediante formulario.
3. **Citas:** Gestión de solicitudes para consultas personalizadas.

##Funcionalidades implementadas
- [x] Catálogo dinámico con conexión a BD.
- [x] Etiquetas visuales por origen del vehículo.
- [x] Formulario de captación de clientes funcional.
- [x] Panel de administración para visualizar leads.
